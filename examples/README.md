# Sephirah Examples

These examples are intentionally small. Their purpose is to show syntax and validation behavior quickly.

## Active examples

- `01-basic-expressions.seph` — arithmetic expressions.
- `02-definitions-and-variables.seph` — definitions, variables, and evaluation.
- `03-declarative-ordering.seph` — declarations and statements in mixed order.

## Future examples

- `future/boolean-return-target.seph.disabled` — first-class boolean return values. Keep disabled until the implementation fully supports boolean values as evaluable/returnable values.

## Notes

If the exact grammar changes, update the examples first. The examples are the public face of the language.
